🚀 How It Works
Upload your portrait
Select gender and dynasty
Choose artistic intensity
Click Begin Transformation
AI generates a historical portrait description based on your face + selected dynasty
📌 Key Idea

This project blends:

Historical Chinese fashion research
AI prompt engineering
Image-to-text analysis
Immersive UI/UX storytelling

to create a "time-travel portrait experience".

⚠️ Notes
Requires a valid Gemini API key
Works best with clear face photos
Does not generate images directly — it generates AI portrait descriptions
